    **SystemProtocols.swift** — Append two new protocols following the existing pattern (protocol + production struct):

    1. `VimPathResolving` protocol:
       - `func resolveVimPath() -> String?` — finds vim binary
       - Production impl `ShellVimPathResolver`: runs `/bin/zsh -l -c "which vim"` via Foundation.Process, reads stdout, trims whitespace, returns path or nil. Discard stderr via separate Pipe(). Per D-08: this respects the user's full shell PATH including Homebrew.

    2. `FileModificationDateReading` protocol:
       - `func modificationDate(of url: URL) -> Date?`
       - Production impl `SystemFileModificationDateReader`: uses `FileManager.default.attributesOfItem(atPath: url.path)[.modificationDate] as? Date`. Per D-05.

    **VimSessionManager.swift** — Create the core @MainActor class:

    1. `enum VimExitResult { case saved, aborted }` — at file scope

    2. `@MainActor final class VimSessionManager` with:
       - Dependencies injected via init: `vimPathResolver: VimPathResolving`, `fileModDateReader: FileModificationDateReading`, with default production values
       - Instance properties (retained to prevent ARC release per Phase 1 pattern):
         - `private var panel: VimPanel?`
         - `private var terminalView: LocalProcessTerminalView?` (import SwiftTerm)
         - `private var continuation: CheckedContinuation<VimExitResult, Never>?`
         - `private var currentTempFileURL: URL?`
         - `private var mtimeBefore: Date?`
       - UserDefaults key `"VimWindowFrame"` for size persistence (D-02)
       - Default window size: 100 columns x 35 rows (Claude's discretion per research)

    3. `func openVimSession(tempFileURL: URL) async -> VimExitResult`:
       - Resolve vim path via `vimPathResolver.resolveVimPath()`. If nil, show NSAlert per D-09 ("vim not found, install via Homebrew"), return `.aborted`.
       - Record mtime before: `mtimeBefore = fileModDateReader.modificationDate(of: tempFileURL)`
       - Store `currentTempFileURL = tempFileURL`
       - Create VimPanel:
         ```swift
         let panel = VimPanel(
             contentRect: centeredRect(on: NSScreen.main ?? NSScreen.screens[0]),
             styleMask: [.titled, .closable, .resizable],
             backing: .buffered,
             defer: false
         )
         panel.level = .floating
         panel.isReleasedWhenClosed = false  // Pitfall 6
         panel.title = "AnyVim"
         ```
       - Restore saved frame from UserDefaults if exists, otherwise use centeredRect (D-01, D-02)
       - Create LocalProcessTerminalView with frame = panel.contentView bounds
       - Set `terminalView.font = NSFont.monospacedSystemFont(ofSize: 13, weight: .regular)` (D-10)
       - Call `terminalView.configureNativeColors()` for system dark/light mode (D-11)
       - Add terminalView to panel.contentView (MUST be added BEFORE startProcess per Pitfall 7)
       - Set `terminalView.processDelegate = self`
       - Use `withCheckedContinuation` to suspend:
         ```swift
         return await withCheckedContinuation { cont in
             self.continuation = cont
             terminalView.startProcess(
                 executable: vimPath,
                 args: [tempFileURL.path],
                 environment: nil,  // SwiftTerm provides TERM=xterm-256color, HOME, etc.
                 execName: nil,
                 currentDirectory: nil
             )
             panel.makeKeyAndOrderFront(nil)
             NSApp.activate(ignoringOtherApps: true)
         }
         ```
       - Store panel and terminalView as instance properties

    4. Implement `LocalProcessTerminalViewDelegate`:
       - `processTerminated(source:exitCode:)`:
         - Compare mtime: `let mtimeAfter = fileModDateReader.modificationDate(of: currentTempFileURL!)`
         - Result: if `mtimeAfter != mtimeBefore` then `.saved` else `.aborted` (D-05, D-06)
         - Save window frame to UserDefaults before closing (D-02)
         - Close panel: `panel?.close()`
         - Clean up: set panel, terminalView, currentTempFileURL to nil
         - Resume continuation: `continuation?.resume(returning: result); continuation = nil`

    5. Handle close button (D-07): Observe `NSWindow.willCloseNotification` on the panel. If the panel closes while continuation is still active (vim still running), the terminal process will be killed by SwiftTerm when the view is removed. Treat as abort — resume continuation with `.aborted`.

    6. Private helper `centeredRect(on screen: NSScreen) -> NSRect`:
       - Use `NSFont.monospacedSystemFont(ofSize: 13, weight: .regular).boundingRectForFont` to calculate character cell size
       - Width = 100 * cellWidth + 20 (padding), Height = 35 * cellHeight + 20 (padding) + ~28 (title bar)
       - Center on screen.visibleFrame

    - `testOpenVimSessionReturnsAbortedWhenVimNotFound` — mock resolver returns nil, verify .aborted result and no crash

    Follow existing test patterns from HotkeyManagerTests.swift and ClipboardGuardTests.swift for mock structure.
